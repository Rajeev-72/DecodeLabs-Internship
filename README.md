# DecodeLabs Internship Projects

This repository contains all my Python development projects built during the DecodeLabs Internship.

## Weekly Projects Index

### 📅 [Week 1: To-Do List Application](./week1)
* **Goal**: Learn fundamental in-memory data management, list storage, manipulation using `append()`, and iterational display.
* **Concepts**: Python Lists, Dictionaries (database rows), Loops, IPO model, and data persistence.
* **Status**: Completed and Tested.

### 📅 [Week 2: Expense Tracker](./week2)
* **Goal**: Learn state accumulation, continuous execution cycles, defensive input validation, and decoupled architectures.
* **Concepts**: Volatility, Accumulator Pattern (`total += expense`), Try-Except blocks, Sentinel Shutdowns (`quit`).
* **Status**: Completed and Tested.

### 📅 [Week 3: Random Password Generator](./week3)
* **Goal**: Learn standard library integration, cryptographic security, Shannon Entropy, and O(N) string joining.
* **Concepts**: secrets module, string module, O(N) memory joining, Shannon Entropy math, and NIST SP 800-63-4 standards.
* **Status**: Completed and Tested.
